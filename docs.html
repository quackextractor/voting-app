6. API Endpointy

Metoda URL Endpoint Popis a tělo požadavku/odpovědi
GET /api/results Vrátí aktuální stavy hlasování. Není potřeba žádná autorizace.
Odpověď: { "options": [{ "id": "a", "votes": 15 }, ...] }
POST /api/vote Uloží nový hlas uživatele.
Tělo: { "optionId": "b" }
Odpověď: { "success": true, "options": [...] }
POST /api/reset Vynuluje všechny hlasy, pokud se token shoduje s konfigurační proměnnou serveru.
Tělo: { "token": "super_tajny_klic_123" }
Odpověď úspěch: { "success": true, "message": "Anketa resetována" }
Odpověď chyba: { "success": false, "message": "Neplatný token" } (Status 403)
